The Mission

기존 스타벅스 페이지를 실제 홈페이지로 효과를 넣어서 작업하기로 하기로 하였다. 스크롤시 올라오는 이미지 등 효과를 넣고 바꿀 수 있는 부분은 적절하게 더 나은 방향으로 바꿔서 작업하였고 실제 홈페이지에서 고쳐야 할 부분들을 체크하고 좀더 사용자에게 편한 방법으로 개선하여 작업 하였다.

The Client

Frontend Class Project

Responsive

PC에서와 비슷한 느낌으로 전체적인 레이아웃을 변경하여 작업하였다. 스크롤에따라 애니메이션이 되며 스크롤 한번에 모든 애니메이션이 끝나도록 하였다. 상단메뉴는 우측의 메뉴버튼을 누르면 나타나고, 메뉴들은 스크롤이 되도록 만들었다.
